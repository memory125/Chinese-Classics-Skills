# 周易学习技能 📜

> 从零开始学习周易，包括入门知识、六十四卦查询、卦辞爻辞解读、起卦解卦、案例分析和每日练习

---

## 🎯 核心功能

### 1. 六十四卦查询
快速查询任意卦象的完整解读，包括：
- **卦象结构**：上下卦、卦德、卦象
- **卦辞爻辞**：原文 + 白话解读
- **多维启示**：修心、事业、健康、人际关系
- **实际应用**：现代场景适配建议

### 2. 起卦解卦
支持多种起卦方式：
- 📅 **每日一卦**：根据日期自动生成（农历 + 节气）
- 🎲 **随机起卦**：快速随机获取一卦
- 🔢 **数字起卦**：输入任意数字起卦
- 🪙 **硬币起卦**：模拟传统硬币法
- ⏰ **时间起卦**：基于当前时间
- ☀️ **节气起卦**：结合 24 节气

### 3. 练习测验系统 ⭐
- **八卦识别测验**（8 题）：卦形、元素、特性识别
- **卦名记忆测验**（12 题）：卦辞、核心精神记忆
- **爻辞理解测验**（5 题）：经典爻辞含义理解
- **案例分析测验**（3 题）：场景适配分析
- **综合测验**（28 题）：全量完整测试
- **成绩保存**：JSON 格式保存历史记录，支持进度追踪

### 4. 基础知识学习
- 八卦基础概念（乾、坤、震、巽、坎、离、艮、兑）
- 阴阳五行理论
- 天干地支与农历转换
- 24 节气与卦象对应

### 5. 案例库
包含典型应用场景：
- **工作决策**：是否接受 Offer、创业时机选择、职业转型风险评估
- **人际关系**：处理团队冲突、同事关系紧张化解、朋友借钱不还处理
- **投资理财**：大额投资时机判断、股票止损决策
- **感情婚姻**：异地恋是否继续、婚前犹豫分析、处理婆媳关系
- **健康养生**：健身计划坚持困难、是否接受手术
- **学习规划**：是否考研深造、学习新技能的选择

### 6. 修心指南 ⭐
- **18 个核心卦象**：8 个经卦 + 10 个常用卦
- **心性修炼**：将卦象智慧转化为日常修行
- **实践指导**：具体可操作的修心方法

## 🔧 工具说明

### main.py ⭐
交互式主入口，带完整菜单系统

```bash

# 提供：每日一卦、交互起卦、练习测验、农历查询、
```

### daily-gua.py
生成每日卦象，结合农历、节气、干支

```bash
python3 daily-gua.py
# 可选参数：
python3 daily-gua.py --date 2024-03-15
python3 daily-gua.py --format json
```

### interactive-gua.py ⭐
交互式起卦解卦系统

```bash
python3 interactive-gua.py
# 支持：硬币法、数字法、时间法
# 包含：问题梳理、自动起卦、初步解读
```

### exercise-system.py ⭐
练习测验系统

```bash
python3 exercise-system.py
# 包含：八卦识别、卦名记忆、爻辞理解、案例分析
# 成绩自动保存至 JSON 文件
```

### get-lunar-ganzhi.py
公历 ↔ 农历转换，干支计算

```bash
python3 get-lunar-ganzhi.py
# 输出：日期、农历、干支、节气
# 支持 2024-2030 年
```

### get-solar-term.py
24 节气查询与计算

```bash
python3 get-solar-term.py
# 输出：当日节气、距离下一节气天数
# 包含：节气卦象推荐
```

### zhouyi_database.py
六十四卦完整数据库，支持程序化查询

```python

# 查询乾卦
```

## ❓ 常见问题

### Q: 这个技能适合什么样的人？
**A**: 适合对传统文化感兴趣、想了解周易但不想啃古文的初学者。零基础友好，所有解读都是白话 + 实用建议。

### Q: 需要背诵所有内容吗？
**A**: 不需要。重点是理解逻辑 + 日常使用。数据都在 `references/` 里，随时可查。

### Q: 每日一卦准确吗？
**A**: 卦象是算法生成的，"准确"在于你如何理解和应用。重点是通过卦象反思当日状态，不是预测未来。

### Q: 可以自定义起卦方式吗？
**A**: 可以。`scripts/` 目录下的脚本都是开源的，你可以修改或扩展。

### Q: 如何更新卦象数据？
**A**: 数据文件在 `references/六十四卦终极版_v10.md`，如需更新请修改后重新加载技能。

### Q: 练习测验的成绩会保存吗？
**A**: 会。成绩自动保存为 JSON 格式，可以查看历史进度。
