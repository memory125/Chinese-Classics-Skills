# 📜 周易学习 Skill v1.0

> **从零开始系统学习周易的实用工具**

一个面向个人学习者的周易学习平台，采用"白话 + 实用"的教学方式，帮助用户从零开始系统掌握周易知识。

---

## 🎯 核心功能

### 1. 六十四卦查询 ⭐
- ✅ **完美数据**: 64 卦完整（无重复、无缺失）
- ✅ **卦象结构**: 上下卦、卦德、卦象符号
- ✅ **原文 + 白话**: 卦辞爻辞对照解读
- ✅ **多维启示**: 修心、事业、健康、人际关系
- ✅ **场景应用**: 现代生活实际案例

### 2. 起卦解卦系统 ⭐
支持三种传统起卦方式：
- 🪙 **硬币法**: 模拟传统三枚硬币抛掷
- 🔢 **数字法**: 输入任意数字生成卦象
- ⏰ **时间法**: 基于当前时间自动起卦

### 3. 练习测验系统 ⭐
- 🧩 **八卦识别测验** (8 题)
- 📝 **卦名记忆测验** (12 题)
- 💬 **爻辞理解测验** (5 题)
- 🎯 **案例分析测验** (3 题)
- 🏆 **综合测验** (28 题)
- 💾 **成绩保存**: JSON 格式历史记录

### 4. 每日一卦 ⭐
- 📅 **精准推荐**: 结合农历 + 节气 + 干支
- 🌞 **节气对应**: 惊蛰→震卦、春分→巽卦等
- 💡 **核心启发**: 根据季节特点提供指导

### 5. 基础知识学习
- 🔮 **八卦基础**: 乾、坤、震、巽、坎、离、艮、兑
- ⚖️ **阴阳五行**: 理论体系与对应关系
- 📅 **天干地支**: 六十甲子纪年法

### 6. 案例库 (18+ 真实案例)
涵盖工作决策、人际关系、投资理财、感情婚姻、健康养生、学习规划六大类。

### 7. 修心指南 (18 个核心卦象)
将卦象智慧转化为日常修行实践指导。

## ❓ 常见问题

### Q: 这个技能适合什么样的人？
**A**: 适合对传统文化感兴趣、想了解周易但不想啃古文的初学者。零基础友好，所有解读都是白话 + 实用建议。

### Q: 需要背诵所有内容吗？
**A**: 不需要。重点是理解逻辑 + 日常使用。数据都在 `references/` 里，随时可查。

### Q: 每日一卦准确吗？
**A**: 卦象是算法生成的，"准确"在于你如何理解和应用。重点是通过卦象反思当日状态，不是预测未来。

### Q: 可以自定义起卦方式吗？
**A**: 可以。`scripts/` 目录下的脚本都是开源的，你可以修改或扩展。

### Q: 练习测验的成绩会保存吗？
**A**: 会。成绩自动保存为 JSON 格式，可以查看历史进度。
