# 起卦与解卦流程指南

## 📌 一、起卦前的准备工作

### 1. 明确问题
在起卦前，请确保：
- 问题具体明确（不要问"我未来怎么样"）
- 有清晰的决策场景（如"A 工作还是 B 工作"）
- 心态平和，不带强烈情绪

**好问题示例：**
- ✅ "我应该接受 A 公司 Offer 还是继续找机会？"
- ✅ "这个月投资计划是否应该执行？"
- ❌ "我运气好不好？"（太模糊）

### 2. 专注与静心
- 找个安静的环境
- 深呼吸几次
- 心里默念问题 3 遍

---

## 📌 二、起卦方法

### 方法 A：硬币法（推荐新手）

**准备：**
- 3 枚相同硬币（任意一元硬币即可）
- 一张纸和笔

**步骤：**
1. 双手合握硬币，静心默念问题
2. 摇晃约 10-20 秒
3. 抛出硬币，记录结果
4. 重复 6 次，每次画一个爻（从下往上）

**结果记录规则：**
| 正反面组合 | 数量 | 爻的类型 | 符号 |
|-----------|------|---------|------|
| 3 个正面 | 少阴 | - - | 断开的阴爻 |
| 2 正 1 反 | 少阳 | —— | 连续的阳爻 |
| 1 正 2 反 | 老阳 | O | 可变阳爻 |
| 3 个反面 | 老阴 | X | 可变阴爻 |

**说明：**
- "老阳/老阴"是变爻，需要特殊解读
- 记录时从下往上画（第一次抛的结果在最下面）

---

### 方法 B：数字法（快速模拟）

**步骤：**
1. 心中默念问题
2. 随机报出三个 1-99 之间的数字
3. 用以下规则转换：
   - 数字 ÷ 8，取余数（余数为 0 则按 8 算）
   - 第一个数字 = 下卦
   - 第二个数字 = 上卦
   - 第三个数字 ÷ 6，取余数 = 变爻位置

**八卦数字对应：**
| 数字 | 八卦 |
|------|------|
| 1 | 乾 ☰ |
| 2 | 兑 ☱ |
| 3 | 离 ☲ |
| 4 | 震 ☳ |
| 5 | 巽 ☴ |
| 6 | 坎 ☵ |
| 7 | 艮 ☶ |
| 8 | 坤 ☷ |

**示例：**
- 数字：23、57、14
- 23 ÷ 8 = 2...余 7 → 艮卦（下卦）
- 57 ÷ 8 = 7...余 1 → 乾卦（上卦）
- 14 ÷ 6 = 2...余 2 → 第二爻为变爻

结果：山地剥卦，二爻动

## 📌 三、解卦流程

### 第 1 步：识别本卦

根据六个爻的组合确定卦名。例如：
- 下乾 ☰ + 上坤 ☷ = 地天泰

### 第 2 步：读卦辞（整体判断）

卦辞是该卦的总体趋势说明，代表事情的大方向。

**示例：**
```
卦：火天大有
卦辞："元亨"
白话解读：大吉大利，目前处于资源丰富、机会充足的阶段
```

### 第 3 步：看变爻（细节分析）

如果有变爻（老阳/老阴），需要：
1. 找到该爻的爻辞
2. 解读该爻的具体含义
3. 结合问题场景应用

**示例：**
```
乾卦第三爻："君子终日乾乾，夕惕若厉"
白话：白天努力进取，晚上谨慎反思
应用场景：适合提醒自己在事业上升期要保持谦逊和警惕
```

### 第 4 步：看之卦（发展趋势）

变爻变化后形成的新卦叫"之卦"，代表事情的发展趋势。

**示例：**
```
本卦：乾为天 ☰☰
第三爻变阴 → 
之卦：天风姤 ☰☴
解读：从刚健进取的状态，可能转向需要谨慎应对变化的状态
```

---

## 📌 四、解卦原则

### 核心原则

1. **先整体后局部**：先看卦辞大方向，再看爻辞细节
2. **联系现实场景**：把卦象对应到你的具体问题
3. **不绝对化判断**：卦象是趋势参考，不是宿命判决
4. **重视变爻**：有变爻说明事情可能发生变化

### 禁忌

❌ 不要过度解读每一个字  
❌ 不要用一次起卦决定一生大事  
❌ 不要沉迷重复起卦同一问题  
❌ 不要把卦象当绝对预测工具

## 📌 五、完整案例示范

**问题：** "我应该现在跳槽还是继续留在当前公司？"

**起卦结果：**
- 本卦：风水涣（巽上坎下）
- 变爻：第四爻
- 之卦：风山渐

**解卦步骤：**

1. **读本卦卦辞：**
   - 涣："亨。王假有庙，利涉大川，利贞"
   - 白话：通达。需要凝聚人心、统一方向；适合突破困难；坚守正道有利

2. **看变爻爻辞：**
   - 九四："涣其群，元吉"
   - 白话：打破小圈子格局，大吉大利

3. **看之卦趋势：**
   - 渐："女归吉，利贞"
   - 代表循序渐进、稳步发展

4. **综合判断：**
   - ✅ 当前环境可能需要突破（涣卦）
   - ✅ 跳出舒适圈是正确方向（九四爻）
   - ✅ 但新选择要稳扎稳打（渐卦提示）

5. **行动建议：**
   - 可以考虑跳槽或岗位调整
   - 找机会前做好充分准备
   - 不要冲动决定，逐步推进

## 📌 六、每日练习建议

**第一周：熟悉流程**
- 每天随意起一卦（不问具体问题）
- 练习识别卦名和读卦辞
- 记录感受但不强求应用

**第二周：联系生活**
- 每天选一个小问题起卦
- 尝试解读爻辞并应用到当天决定
- 事后复盘验证判断准确性

**第三周及以后：**
- 遇到实际决策时谨慎使用
- 培养自己的"卦感"（直觉理解力）
- 结合现实情况灵活判断

---

> 📖 **重要提醒**：起卦只是思考工具，不是替代你理性判断。最准确的"卦"始终在你自己的心里。
