### 1. RAG 检索增强引擎 (核心功能)

**文件**: `reference/vector_db.py`

**功能**:
- TF-IDF 向量检索
- 智能文本分割
- 相似度匹配
- 案例精准搜索

**实现原理**:
```python
# 文本分割
TextSplitter.split_chinese_text(text, max_length=1000)

# 向量化
TextVectorizer.fit(documents)  # 建立词汇表和 IDF
TextVectorizer.transform(documents)  # TF-IDF 向量

# 检索
engine.search(query, top_k=5)
```
