### 4. 历史知识图谱 (Knowledge Graph)

**文件**: `scripts/knowledge_graph.py`

**功能实现**:
- ✅ 人物 - 事件 - 地点关系图谱
- ✅ 关联路径查找 (多跳关系)
- ✅ 时间线自动生成
- ✅ Mermaid 可视化输出

**核心能力**:
```python
# 实体查询
kg.get_entity_details("刘邦")
# 输出：刘邦人物档案，包含身份、朝代、相关事件

# 关系路径
kg.find_paths("刘邦", "张良")
# 输出：刘邦 -> 张良 / 刘邦 -> 项羽 -> 张良

# 时间线
kg.generate_timeline("刘邦")
# 输出：鸿门宴 -> 楚汉战争 -> 建立汉朝

# 可视化
kg.generate_mermaid_graph(["刘邦", "项羽", "鸿门宴"])
# 输出：Mermaid 流程图代码
```

**测试输出**:
- ✓ 实体详情完整
- ✓ 路径查找准确 (3 条路径)
- ✓ 时间线清晰
- ✓ Mermaid 图生成成功

### 5. 历史推演模式 (Historical Simulation)

**文件**: `scripts/historical_simulation.py`

**功能实现**:
- ✅ 分支情景模拟
- ✅ 概率预测系统
- ✅ 后果推演引擎
- ✅ 历史相似性对比

**核心场景**:
1. **鸿门宴推演**
   - 分支 1: 杀刘邦 (30% 概率)
   - 分支 2: 扣押刘邦 (50% 概率)
   - 历史选择：放走刘邦 (20% 概率)

2. **削藩策略推演**
   - 汉武帝激进削藩 vs 温和削藩
   - 概率分析：40% vs 60%

3. **明朝削藩推演**
   - 建文帝激进 vs 温和策略对比

**推演输出结构**:
```
- 直接后果 (Immediate Consequences)
- 短期结果 (Short-term Outcome)  
- 长期结果 (Long-term Outcome)
- 关键因素 (Key Factors)
- 历史相似 (Historical Similarity)
```

**测试验证**:
- ✓ 鸿门宴分支推演完整
- ✓ 削藩策略对比清晰
- ✓ 概率分布合理
- ✓ 历史相似性准确
