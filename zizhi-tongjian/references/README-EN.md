# 资治通鉴 - 历史索引库

This repository contains reference materials for the Zizhi Tongjian AI Skill.

## 文件说明 / File Structure

### 1. **character-index.md** - 人物索引
- 包含历史人物时间线、核心特质、成功要素、致命伤
- 支持按人物查询履历和分析

### 2. **case-library/** - 案例库
- 包含经典历史案例的完整解析
- 每个案例包含：原文、白话、关键点、现代映射、建议

### 3. **scripts/** - 脚本目录
- **modern_scenario_mapper.py** - 现代场景映射器
- **historical_simulator.py** - 历史沙盘模拟器  
- **character_trajectory.py** - 人物履历生成器
- **style_switcher.py** - 多文风生成器
- **daily_wisdom.py** - 今日锦囊盲盒

## 功能概述 / Features

### 1. 精准史实检索 (Precise Historical Search)
- 294 卷、1362 年历史全覆盖
- 准确出处标注
- 绝对忠于史实

### 2. 现代问题历史解 (Modern Problems, Historical Solutions)
- 职场管理、人际关系、投资理财等场景映射
- A/B Test 对比分析
- 可操作的现代建议

### 3. 人物深度解析 (Character Analysis)
- 完整人物履历表
- 成功要素与致命伤分析
- 现代启示

### 4. 互动体验 (Interactive Experience)
- 历史沙盘模拟器
- 多文风一键切换
- 今日锦囊盲盒

## 使用方式 / Usage

```bash
# 安装依赖
pip install -r requirements.txt

# 测试功能
python scripts/modern_scenario_mapper.py
python scripts/historical_simulator.py
python scripts/character_trajectory.py
python scripts/style_switcher.py
python scripts/daily_wisdom.py
```
