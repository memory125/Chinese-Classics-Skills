# 📚 资治通鉴 Skill (Zizhi Tongjian)

> 基于 OpenClaw 的智能历史学习平台，整合《资治通鉴》294+ 卷原文、200+ 精选案例和 AI 智能搜索系统。提供文言文翻译、人物履历生成、历史沙盘模拟等强大功能。

---

## 🎯 核心功能

### 1. RAG v5.0 智能搜索 ⭐
- **混合搜索**：关键词 + 向量语义双引擎
- **200+ 精选案例库**：从战国到清朝的经典历史事件
- **智能排序**：基于相关性和时间权重
- **上下文增强**：自动关联相似案例

### 2. SQLite 原文数据库
- **294+ 卷完整内容**：覆盖战国至北宋 1362 年历史
- **FTS5 全文检索**：毫秒级查询响应
- **智能分词**：文言文专用分词器
- **版本管理**：支持多版本对比

### 3. AI 文言文翻译引擎
- **原文 + 白话对照**：逐句精准翻译
- **注释解析**：生僻字、典故解释
- **上下文理解**：结合历史背景的智能翻译
- **批量处理**：支持长文本翻译

### 4. 人物履历生成器
- **自动提取**：从原文中提取人物信息
- **时间线梳理**：生平大事记自动生成
- **关联分析**：人物关系网络可视化
- **多维度标签**：政治、军事、文化等分类

### 5. 今日锦囊推荐系统
- **每日推送**：精选历史智慧片段
- **场景适配**：职场、人生、决策等分类
- **智能推荐**：基于用户兴趣的个性化推荐
- **收藏分享**：支持导出和分享

### 6. 历史沙盘模拟器 ⭐
- **15+ 经典事件重演**：鸿门宴、赤壁之战等
- **多结局模拟**：不同决策导致的历史走向
- **交互式体验**：扮演历史人物做决策
- **教育价值**：理解历史因果关系

### 7. FastAPI RESTful API
- **30+ 端点**：搜索、翻译、查询、分析等
- **认证系统**：JWT Token 安全验证
- **文档生成**：Swagger UI + ReDoc
- **异步处理**：支持高并发请求

### 8. Streamlit Web 界面 ⭐
- **7 个功能模块**：智能搜索、AI 对话、人物查询等
- **可视化图表**：知识图谱、时间线、统计图
- **交互式操作**：拖拽、点击、筛选
- **响应式设计**：支持桌面和移动端

## 🚀 快速开始

### 方式一：通过 OpenClaw 技能调用（推荐）

```bash
# 在 OpenClaw 中直接询问
"将相和的故事是什么？"
"分析刘邦的人物履历"
"今天的历史锦囊是什么？"
"模拟鸿门宴，如果我站在项羽角度会怎样？"
"翻译：臣本布衣，躬耕于南阳"
```

### 方式二：使用 RESTful API

```bash
# 启动 API 服务
cd zizhi-tongjian/api
python3 -m uvicorn main:app --reload --port 8002

# 智能搜索案例
curl http://localhost:8002/search/hybrid \
  -H "Content-Type: application/json" \
  -d '{"query": "将相和", "top_k": 5}'

# AI 文言文翻译
curl http://localhost:8002/translate/classical \
  -H "Content-Type: application/json" \
  -d '{"text": "臣本布衣，躬耕于南阳"}'

# 人物履历查询
curl http://localhost:8002/profile/liu-bang \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 方式三：使用 Streamlit Web 界面

```bash
cd zizhi-tongjian
streamlit run web_chat.py --server.port 8501
```

访问 http://localhost:8501，即可通过图形化界面与 AI 助手对话。

### 示例 2：AI 文言文翻译

**问题**： "翻译：臣本布衣，躬耕于南阳"

**回答**：
```
📜 AI 文言文翻译引擎

【原文】臣本布衣，躬耕于南阳

【白话译文】我本来是个平民百姓，在南阳亲自耕种田地。

【字词解析】
- 臣：古代臣子对君主的自称
- 布衣：平民百姓（古代平民穿布衣）
- 躬耕：亲自耕种
- 南阳：地名，今河南南阳一带

【出处】诸葛亮《出师表》

【背景故事】
这是诸葛亮在公元 227 年北伐前写给后主刘禅的奏章。
当时诸葛亮准备出征，临行前回顾自己从布衣到丞相的
经历，表达忠诚和报效之心。

【现代启示】
1. 不忘初心：成功不忘本
2. 谦虚谨慎：即使身居高位也要保持本色
3. 感恩回报：对知遇之恩要铭记于心
```
