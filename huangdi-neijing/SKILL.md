- **饮食劳倦**: 生活方式对健康的影响

- **饮食调理**: 五味入五脏，食疗方剂
- **起居有常**: 作息规律与睡眠质量
- **情志调摄**: 心理健康与情绪管理

### 4. 症状智能分析工具 ⭐
- **头痛分析**: 不同类型头痛的辨证
- **失眠调理**: 入睡困难、早醒等问题

## 💡 使用示例

### 示例 1：症状分析

**问题**: "我最近头痛，失眠，乏力怎么办？"

**回答**:
```markdown
🌿 黄帝内经症状分析

【症状组合】头痛 + 失眠 + 乏力

【可能证型】
1. 肝阳上亢型
   - 特征：头胀痛、急躁易怒、面红目赤
   - 建议：平肝潜阳，菊花茶、天麻炖鸡

2. 气血两虚型
   - 特征：隐痛绵绵、面色苍白、心悸气短
   - 建议：补气养血，八珍汤、黄芪炖鸡

3. 肾精不足型
   - 特征：头晕耳鸣、腰膝酸软、记忆力减退
   - 建议：补肾填精，六味地黄丸、黑芝麻糊

【通用调理】
✅ 保证充足睡眠（23:00 前入睡）
✅ 饮食清淡，少食辛辣
✅ 适度运动，如太极拳、八段锦
✅ 保持心情舒畅，避免焦虑

【建议就医】如果症状持续超过一周，请及时就医。
```

### 示例 2：节气养生

**问题**: "春季如何养生？"

**回答**:
```markdown
🌸 春季养生指南（立春 - 谷雨）

【季节特点】
- 阳气升发，万物复苏
- 肝气旺盛，易怒伤肝
- 风邪当令，易感外邪

【养生原则】
✅ 夜卧早起：晚睡早起，顺应阳气生发
✅ 广步于庭：适度运动，舒展筋骨
✅ 情志舒畅：保持心情愉悦，避免抑郁愤怒
✅ 春捂秋冻：注意保暖，不可过早减衣

【饮食调理】
- 宜食：韭菜、豆芽、菠菜、枸杞、红枣
- 忌食：生冷寒凉、油腻辛辣
- 推荐食疗：枸杞菊花茶、红枣桂圆汤

【穴位保健】
- 太冲穴：疏肝理气，按揉 3-5 分钟
- 足三里：健脾和胃，增强免疫力
- 风池穴：预防感冒，缓解头痛

【运动建议】
✅ 晨起散步、慢跑
✅ 太极拳、八段锦
✅ 放风筝（舒展筋骨）
```

### 示例 3：阴阳五行基础

**问题**: "什么是阴阳五行？"

**回答**:
```markdown
🌿 黄帝内经·阴阳五行基础

【阴阳学说】
阴阳是宇宙万物的基本规律，代表对立统一的两个方面。

- **阳**: 主动、上升、温热、明亮、外向
- **阴**: 主静、下降、寒凉、晦暗、内向

**阴阳关系**:
1. 阴阳对立：相互制约（如寒热）
2. 阴阳互根：相互依存（如无阴则无阳）
3. 阴阳消长：此消彼长（如昼夜交替）
4. 阴阳转化：物极必反（如重阴必阳）

【五行理论】
五行是木、火、土、金、水五种基本元素及其运动变化。

**五行相生**: 木→火→土→金→水→木
**五行相克**: 木→土→水→火→金→木

**五脏对应**:
- 肝属木：主疏泄，藏血
- 心属火：主血脉，藏神
- 脾属土：主运化，统血
- 肺属金：主气司呼吸
- 肾属水：主藏精，纳气

【临床应用】
通过阴阳五行理论，可以：
1. 诊断疾病（辨证论治）
2. 指导治疗（调整平衡）
3. 预防养生（未病先防）
```
